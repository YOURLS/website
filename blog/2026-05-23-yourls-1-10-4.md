We've just released [YOURLS 1.10.4][the release page]!

[the release page]: https://github.com/YOURLS/YOURLS/releases/tag/1.10.4

<!--truncate-->

## What's new?

Filter improvements, a security patch, and more!

- A cross-site scripting vector in stats pages has been fixed ([advisory](https://github.com/YOURLS/YOURLS/security/advisories/GHSA-5h77-88j3-r659))
- Date and time display can now be localized
- Debug functions and logic updated
- Plugins can now filter SQL queries

Filtering SQL queries can be used to create plugins for other SQL dialects. For example, see @[ozh](https://github.com/ozh)'s [new PostgreSQL drop-in plugin](https://github.com/ozh/yourls-postgresql) and his personal [blog post about it](https://planetozh.com/blog/2026/05/yourls-now-speaks-postgresql/).

We're especially grateful to @[sondt99](https://github.com/sondt99) for reporting the security issue with referrers being displayed in stat pages.
This, once again, shows the benefit of open-source code!

These are just the highlights; see [the release page][] or our full [CHANGELOG](https://github.com/YOURLS/YOURLS/blob/master/CHANGELOG.md) for more details.

## How to update?

It's the typical case of "upload files over older ones and forget", which as always doesn't affect your plugins and user config.

Also as always, if it's been a while since your last DB backup, we suggest you do that **before** updating, since the new version will ask your database to change some stuff.
